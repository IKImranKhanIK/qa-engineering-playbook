# Code of Conduct

## Our Standards

This project maintains a professional environment focused on quality engineering education and collaboration.

### Expected Behavior
- Use professional language
- Respect different viewpoints and experiences
- Accept constructive criticism
- Focus on technical merit
- Help others learn and grow

### Unacceptable Behavior
- Personal attacks or insults
- Harassment of any kind
- Publishing others' private information
- Trolling or deliberately disruptive behavior
- Off-topic discussions that derail technical conversations

## Scope

This Code of Conduct applies to:
- Repository interactions (issues, pull requests, discussions)
- Communication channels related to this project
- Representing the project in public spaces

## Enforcement

### Reporting
Report violations by:
- Opening a private issue with the `code-of-conduct` label
- Contacting project maintainers directly

### Response
Maintainers will:
- Review reports promptly
- Investigate thoroughly
- Maintain confidentiality
- Take appropriate action

### Consequences
Violations may result in:
- Warning
- Temporary ban from project interactions
- Permanent ban from the project

## Attribution

This Code of Conduct is focused on maintaining a productive, professional environment for learning and collaboration in quality engineering.
